# Inspiration

This project explores the idea that AI assistants become more useful when they
can recover durable, source-backed context across sessions.

Useful ingredients:

- Knowledge graphs for explicit concepts and relationships.
- Retrieval for recall.
- Provenance for trust.
- Review/eval loops for quality control.
- Offline-first tooling for private data.

The public repository keeps this inspiration generic. Private notebooks,
conversation exports, and owner-specific context should be ingested into a local
graph outside git.
